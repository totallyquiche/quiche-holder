# Quiche Holder

A simple [Lumen](https://lumen.laravel.com/) API for generating placeholder images of the wonderfully versatile, always delectable quiche!

## Usage

Make a GET request to `https://[APPLICATION URL]/[WIDTH]/[HEIGHT]`
